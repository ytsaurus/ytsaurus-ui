The files in json folder might be generated by commands below:

```bash
# cd https://github.com/ytsaurus/ytsaurus/tree/main/yt/admin/dashboards
$ cd /path/to/github/ytsaurus/yt/admin/dashboards && ya make
$ yt_dashboards/bin/generate_dashboards list | grep grafana | cut -d ' ' -f 1 \
    | xargs -I {} bash -c "
        echo {};
        yt_dashboards/bin/generate_dashboards json {} --backend grafana > /path/to/github/ytsaurus-ui/packages/ui/tests/monitoring/json/{}
    "
```
